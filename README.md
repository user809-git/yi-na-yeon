## 2. 활용 비율 보기
### 2.1 리스트 탭
- 탭들
  - (Adobe) Photoshop  
  - (Adobe) illustrator
  - HTML5
  - CSS3
  - 반응형 웹
  - JavaScript(ES6+)
  - jQuery
  - JSP
  - Servelet
  - Oracle Database 
* _Internet Explorer에서는 항목들을 탭했을 때 하나씩 표시되지 않고 탭의 배경에 비율만큼 채워져서 보이도록 함_
### 2.2 항목 별 비율 표시 바(bar)
* 배운 언어들 중 활용 빈도 수나 활용가능한 정도를 고려하여 100% 중에 활용 비율을 표시
* _Internet Explorer에서 작동 되지 않음 (비율을 탭에 직접 표시_)
